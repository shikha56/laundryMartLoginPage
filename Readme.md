# Laundry Wallah / Laundry Mart Login Page Using Flex

A simple and responsive **HTML webpage** for **Laundry Mart Login Page Using Flex**.  
A sleek, lightweight, and modern Login interface designed for Laundry Mart, a digital laundry and dry-cleaning service application. This repository contains a pure front-end implementation using only HTML5 and CSS3 (No JavaScript).

## 🛠️ Technologies Used

- **HTML:** Semantic layout structuring (<form>, <input>, <main>, <section>).
- **CSS:** Flexbox .
